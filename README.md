# 卡卡利科夜未眠 · 胜率提升工具

《血染钟楼》自制剧本 **《卡卡利科夜未眠》**（作者：海伦娜 & 纾）的单页交互式实战工具。

> 目标不是百科，而是帮玩家在实战中 **30 秒内拿到一个更高胜率的行动建议**。

## 在线访问

👉 https://boomlulu.github.io/Blood-on-the-Clocktower/

## 功能（5 个核心 Tab）

| Tab | 作用 |
| --- | --- |
| 🎭 **我的角色** | 选人数看配置 → 选角色，查看实战打法卡（核心任务 / 是否公开 / 信息方式 / 夜间目标 / 配合 / 最怕反制 / 常见错误 / 高胜率一句话）。25 个角色，21 个重点角色含完整策略。 |
| 🔍 **当前局势** | 7 种实战局势，输出原因排序、该问什么、不要做什么、优先怀疑谁、用谁验证。 |
| ⚔️ **组合反制** | 6 个高胜率核心邪恶组合：威胁点 / 识别信号 / 善良方反制 / 邪恶方技巧 / 常见误判。 |
| 🎯 **今天怎么打** | 回答 7 个问题，生成今日决策建议（怀疑机制 / 不该做 / 该问 / 处决类型 / 保护类型）。 |
| 🌃 **夜晚顺序** | 首夜 / 其他夜行动顺序，点击展开每步实战提醒。 |

## 技术

- 单文件 `index.html`，纯 HTML/CSS/JS，无任何依赖与后端
- 移动端友好，暗色背景 + 羊皮纸卡片风格
- 所有数据用 JS 对象数组维护，增改角色/建议直接改数组即可

## 文件说明

- `index.html` / `卡卡利科夜未眠.html` — 网页（同一份内容）
- `卡卡利科夜未眠文字整理版.md` — 剧本角色与流程整理
- `善良阵营打法攻略.md` / `邪恶阵营打法攻略.md` — 攻略源文档
